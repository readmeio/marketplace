# StatusPage

## Overview

A `<StatusPage />` can be used to display the current health, uptime, or incident history of your services or systems.

<img src="status-page.png" width="800" />

## Usage

```mdx
<StatusPage key="123" />
```

## Props

| Prop  | Type   | Description      |
| ----- | ------ | ---------------- |
| `key` | number | The status code. |
